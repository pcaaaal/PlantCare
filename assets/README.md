# Asset Platzhalter

Diese Datei zeigt an, dass Standard-Assets von Expo bereitgestellt werden.

## Benötigte Assets

Die folgenden Asset-Dateien werden von der App erwartet:

- `icon.png` - App-Icon (1024x1024 px)
- `splash.png` - Splash Screen (1284x2778 px für iOS)
- `adaptive-icon.png` - Android Adaptive Icon (1024x1024 px)
- `favicon.png` - Web Favicon (48x48 px)

## Assets erstellen

Du kannst eigene Assets erstellen oder die Standard-Assets von Expo verwenden.

Wenn du die App mit `npm start` ausführst, werden fehlende Assets durch Expo-Standard-Assets ersetzt.

## Eigene Assets hinzufügen

1. Erstelle die benötigten Bilddateien
2. Platziere sie in diesem `assets/` Verzeichnis
3. Aktualisiere `app.json` bei Bedarf mit den richtigen Pfaden
